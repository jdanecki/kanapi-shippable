How to create new docker APPS image automatically
=============================================

1. To create APPS docker image run: make_image

How to use BBS docker to build APPS image manually
=======================================================

1. In interactive directory run
./create_docker
./run_docker
2. In running container:
/kanapi_build/kanapi/kanapi_system/docker/configure
cd /kanapi_build/kanapi/kanapi_system/kanapi-apps
./install apps verbose
