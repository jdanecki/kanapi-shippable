How to create new docker X11 image automatically
=============================================

1. To create X11 docker image run: make_image

How to use CLI docker to build X11 image manually
=======================================================

1. In interactive directory run
./create_docker
./run_docker
2. In running container:
/kanapi_build/kanapi/kanapi_system/docker/configure
cd /kanapi_build/kanapi/kanapi_system/kanapi-x11
./install x11 verbose
