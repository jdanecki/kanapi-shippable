How to create new docker images
===============================

1. To create SBS docker image see README.txt in sbs directory
2. To create BBS docker image see README.txt in bbs directory
3. To create CLI docker image see README.txt in cli directory
4. To create X11 docker image see README.txt in x11 directory
5. To create GUI docker image see README.txt in gui directory
6. To create APPS docker image see README.txt in apps directory
